  • $\displaystyle{\int \sec^{2} x \ dx = }$ $+C$

    Hint

    What function do we differentiate to get $\sec^{2} x$ ?

    Show/Hide Solution

    We know that $$\frac{d \tan x + C}{dx} = \sec^{2} x.$$ Therefore, $$\int sec^{2} x dx = \tan x +C.$$
