    Enabling Github

    1.  Go to Github, and go to Settings > Developer Settings > Personal Access Token
    2. Generate an access token with repository access. This means checking the first box, which will automatically check other boxes.
    3. Follow the instructions given to you to attach this token to your Course Maker account. (Yes, you need a separate account.)
    Note: You may write courses before enabling Github, but you must have a Course Maker account.
